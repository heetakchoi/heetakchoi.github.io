숫자 넣어 때려 맞추기

1/26/ 2010
누군가 a t + b sin( c t ) = d 의 해를 어떻게 구할까 라고 질문을 올렸다.
허리도 아프고 퇴근시간도 거의 다 되었고 해서 잠시 생각해 보았다.
at + bsin(ct) = d
bsin(ct) = d - at
sin(ct) = (d - at) / b
Therefore, |(d-at)/ b| <= 1
-1 <= (d - at)/b <= 1
if b>0
-b <= d - at <= b
-b + d <= at <= b + d
if a>0
(-b + d)/a <= t <= (b + d)/a
여기서 f(t) = at + bsin(ct) -d 라고 하고
N 개로 잘라내서 f(t) 의 부호가 바뀌는 걸 확인한다면,
전체 길이가 (b + d)/a - (-b + d)/a = 2b /a
따라서 단위 t 의 길이는 ( 2b / a )/N
